# Fresh Curator（生鲜管家）微信小程序

> 一款有机生鲜食品电商小程序，基于原始 HTML 设计完整还原开发

## 📱 功能概览

| 页面 | 功能 |
|------|------|
| **首页** | 搜索栏、会员横幅、分类网格、配送方式卡片、店长精选商品列表、加入购物车 |
| **分类页** | 左侧分类导航栏、右侧商品瀑布流、分类切换、商品详情跳转、加购 |
| **购物车页** | 商品列表/空状态切换、全选/单选、数量增减、删除商品、实时总价结算 |
| **我的页面** | 微信登录/退出、订单状态查看、收货地址管理、积分查看、优惠券、联系客服、社群二维码 |
| **商品详情** | 图片轮播、规格选择、收藏功能、加购/立即购买、配送信息 |
| **订单确认** | 地址选择、支付方式、价格明细、提交订单 |

## 🚀 完整部署教程

### 第一步：准备工作

#### 1. 注册微信小程序账号
1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「注册」→ 选择「小程序」账号类型
3. 使用未注册过公众号/小程序的邮箱完成注册
4. 登录后进入「设置」→「开发者设置」，获取 **AppID**

#### 2. 安装微信开发者工具
1. 下载地址：[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 根据你的操作系统选择对应版本下载安装
3. 安装完成后使用微信号扫码登录

---

### 第二步：导入项目

1. 打开微信开发者工具
2. 点击左上角 **「+」号** 或选择「项目」→「导入项目」
3. 填写以下信息：
   - **项目目录**：`c:\Users\Administrator\Desktop\stitch\miniprogram`
   - **AppID**：填入你申请的 AppID（或使用测试号）
   - **项目名称**：`Fresh Curator`
4. 点击「确定」完成导入

> **注意**：首次运行可能需要勾选「不校验合法域名」（在右上角详情 → 本地设置中）

---

### 第三步：配置 AppID

打开 `miniprogram/app.json`，确认基础配置正确：

```json
{
  "pages": [
    "pages/index/index",
    "pages/category/category", 
    "pages/cart/cart",
    "pages/me/me",
    "pages/product-detail/product-detail",
    "pages/order/order",
    "pages/address/address"
  ],
  "tabBar": {
    "custom": true,
    "list": [...]
  }
}
```

**关键点**：
- `tabBar.custom: true` 表示使用自定义 TabBar 组件
- 自定义 TabBar 位于 `custom-tab-bar/index` 目录下

### 第五步：发布上线

#### 5.1 上传代码
1. 点击工具栏 **「上传」** 按钮
2. 填写版本号（如 `1.0.0`）和备注
3. 上传成功后进入微信公众平台后台

#### 5.2 提交审核
1. 登录 [微信公众平台](https://mp.weixin.qq.com/) 
2. 进入「版本管理」→「提交审核」
3. 填写以下信息：
   - **功能页面**：填写各页面路径和说明
   - **测试账号**：提供可用的测试账号信息
   
4. 提交后等待微信审核（通常 1-3 个工作日）

#### 5.3 审核通过后发布
- 审核通过后在「版本管理」中点击「发布」

## 🔧 核心技术实现

### 数据存储方案
```javascript
// app.js 中全局数据
globalData: {
  cartItems: [],       // 购物车数据（同步到 wx.setStorageSync）
  userInfo: null,      // 用户信息（登录后保存到 wx.storage）  
  products: [...],     // 首页商品数据
  hotpotProducts: [...] // 分类页火锅商品
}

// 购物车持久化
wx.setStorageSync('cartItems', this.globalData.cartItems)
```

### 自定义 TabBar
```json
// app.json 中声明
"tabBar": { "custom": true, ... }
// custom-tab-bar/index.js 自动根据当前页面高亮对应Tab
```

### 关键 API 使用
| 功能 | API | 说明 |
|------|-----|------|
| 登录 | `wx.getUserProfile` | 获取用户头像昵称 |
| 收货地址 | `wx.chooseAddress` | 调用系统地址簿 |
| 拨打电话 | `wx.makePhoneCall` | 联系客服 |
| 分享 | `onShareAppMessage` | 分享给好友获取积分 |

## ⚠️ 注意事项

1. **图片资源**：当前使用的是外部 CDN 图片链接，生产环境建议上传至微信云存储或自有服务器
2. **域名白名单**：正式发布前需要在微信公众平台配置合法的 request 域名
3. **隐私协议**：需要在小程序后台完善《用户隐私保护指引》
4. **登录限制**：`getUserProfile` 需要在真机上测试才能正常弹出授权窗口
5. **性能优化**：建议对长列表使用虚拟滚动（当前数据量较小暂不需要）
