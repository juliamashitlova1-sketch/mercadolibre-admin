## 📞 FAQ - 常见问题

### Q: 为什么选择 #0052CC 作为主色？
A: 这是一个经典的企业蓝色，具有：
- 高可信度和专业感
- 良好的对比度（符合WCAG AA标准）
- 在各种背景下都清晰可见
- 与电商/数据分析领域的惯例一致

### Q: 可以自定义颜色吗？
A: 可以。所有颜色都存储在 CSS 变量和 Tailwind 配置中，修改一处即可全局应用。

### Q: 动画是否会影响性能？
A: 不会。使用的都是 CSS 动画（60fps），比 JavaScript 动画效率高得多。

### Q: 可以在移动端上使用吗？
A: 完全支持。设计系统包含 4 个响应式断点，确保各种设备上的显示效果。

### Q: 如何添加新的组件？
A: 创建新组件时遵循以下规则：
1. 使用设计令牌（颜色、间距、圆角等）
2. 支持 Variant Props (主/次/等)
3. 支持 size Props (sm/md/lg)
4. 添加必要的状态 (hover/focus/disabled)

## 💡 最佳实践建议

### DO ✅
- 使用 CSS 变量和 Tailwind 类而不是硬编码颜色
- 始终为交互元素提供反馈
- 保持一致的间距和对齐
- 优先使用现有组件而不是创建新的
- 定期检查颜色对比度

### DON'T ❌
- 不要混用不同系列的颜色
- 不要使用不在规范中的颜色值
- 不要省略 hover/focus 状态
- 不要使用随意的圆角和阴影
- 不要在响应式上妥协
